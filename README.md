# 🔬 Motion & Relative Velocity Lab

The **Motion & Relative Velocity Lab** is a free, interactive physics simulator built for science and math classrooms, available online at **[rajat-ed.github.io/motionlab](https://rajat-ed.github.io/motionlab)**. It puts a large, smooth animation track in the exact center of the screen to help students clearly see how things move in real time. In **Standard Mode**, teachers can connect up to five different sections of constant speed or acceleration, using colored track blocks to show exactly where a new motion begins. In **Relative Velocity Mode**, students can click and drag two different characters anywhere on the track, type in positive or negative speeds to change their directions, and see the exact relative velocity calculated instantly on a simple digital dashboard. Because the entire application is contained inside a single, lightweight file with no extra images or code to download, it works perfectly on any computer or tablet and can be set up on GitHub Pages in just a few clicks.
